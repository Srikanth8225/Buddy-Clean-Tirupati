# **App Name**: Buddy Clean

## Core Features:

- Homepage with CTAs: Visually appealing hero section with CTAs to 'Book Home Cleaning' and 'Book Car Wash'.
- Service Pages: Dedicated pages for home cleaning and car wash services, displaying service details and pricing variants in a card layout.
- Shopping Cart: Allow users to add services with selected variants, adjust quantities, remove items, and proceed to checkout.
- Phone Authentication: Implement secure login/signup using phone number and OTP verification via 2Factor.in.
- Checkout and Booking: Multi-step checkout process including contact information, service address with Google Maps Places API autocompletion and validation, and a date/time selector.
- Payment Integration: Integrate Razorpay for online payments and offer a 'Cash on Delivery' option.
- Order History: Private page for logged-in users to view their past and current orders.
- Admin Panel Authentication: Secure admin panel access restricted to authorized users based on a Firestore whitelist.
- Admin Dashboard: Admin overview page with key business metrics and management pages for orders, services, and customers.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) to evoke cleanliness and freshness.
- Background color: Light off-white (#F7F7F2), subtly tinted with the primary hue.
- Accent color: Earthy orange (#DFA262) for calls to action and important highlights.
- Body and headline font: 'PT Sans' for a modern and clean user experience.
- Use line icons related to cleaning and car wash services.
- Card-based layout for service listings to provide clear service and pricing information.
- Subtle animations on button hover states and transitions between pages.

# Buddy Clean - Cleaning Services in Tirupati

A professional web application for a cleaning service provider, offering home cleaning and car wash services in the Tirupati region.

## Features

- **Service Booking:** Browse and book various home and car cleaning packages.
- **Real-time Database:** Powered by Firebase Firestore for orders and service management.
- **Authentication:** Secure phone-based login with Firebase Authentication.
- **Admin Dashboard:** Comprehensive panel for managing orders, services, and users.
- **Responsive Design:** Optimized for all devices using Tailwind CSS and ShadCN UI.
- **Cart & Checkout:** Seamless shopping experience with integrated mock payments.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **UI Components:** ShadCN UI
- **Styling:** Tailwind CSS
- **Backend/Auth:** Firebase
- **AI Integration:** Genkit

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up your Firebase environment variables in a `.env` file.
4. Run the development server: `npm run dev`
